输入限制
属性 描述
disabled 规定输入字段应该被禁用
max 规定输入字段的最大值
maxlength 规定输入字段的最大字符数
min 规定输入字段的最小值
pattern 规定通过其检查输入值的正则表达式
readonly 规定输入字段为只读（无法修改）
required 规定输入字段是必需的（必需填写）
size 规定输入字段的宽度（以字符计）
step 规定输入字段的合法数字间隔
value 规定输入字段的默认值
html5 input 增加属性
属性 描述
autocomplete 规定表单或输入字段是否应该自动完成，适用于text，search，url，tel，email，password，datepickers，range，color
novalidate novalidate属性属于form属性，在提交表单时不对表单数据进行验证
autofocus autofocus是布尔属性，规定加载时应该自动获取焦点
form form属性规定所属一个或多个表单
